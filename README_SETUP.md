# 내가 고른 종목만, 조건 맞을 때만 알림 버전

## Secret
- DISCORD_WEBHOOK_URL = 디스코드 웹훅 링크

## Variables
- PORTFOLIO_TICKERS = AAPL,MSFT,NVDA,005930.KS
- ANALYSIS_PERIOD = 1y
- BUY_THRESHOLD = 7
- SELL_THRESHOLD = 8
- ALERT_MODE = signals_only
- SHORT_ALERT = 1

## 설명
- signals_only : 조건 만족할 때만 알림
- all : 매번 전체 알림
- SHORT_ALERT = 1 : 짧은 알림
- SHORT_ALERT = 0 : 긴 알림

## 폰에서 쓰는 방법
1. 디스코드 앱 설치
2. 알림 켜기
3. 네 서버 채널 고정
4. 홈 화면 바로가기는 브라우저에서 저장소 주소를 추가하면 됨
